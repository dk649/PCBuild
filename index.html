Building The PC

Processor
This is the processor. The Processor is the brain of the computer.
MotherBoard
This is the MotherBoard. The MotherBoard helps connect all the different parts of the computer together. For example the processor, the hard drive and the memory are all connected to the MotherBoard.
Memory
This is the Memory. Computer memory (RAM) is used to store applications we are currently using. Stores information short term.
Hard Drive
This is the Hard Drive. It stores information long term. When you save information it is stored on the Hard Drive.
PowerSupply
This is the PowerSupply. its converts the house electricity into a form of electricity that the computer can use.
